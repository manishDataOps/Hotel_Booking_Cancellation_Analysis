# Hotel_Booking_Cancellation_Analysis
#This project analyzes hotel booking cancellation using SQL

## Project Objective
To analyze hotel booking data and identify key factors responsible for cancellations.

## Tools Used
- SQL (MS SQL Server)

## Key Analysis
- Overall booking vs cancellation rate
- City vs Resort hotel comparison
- Lead time impact on cancellations
- Market segment wise cancellation behavior
- New vs repeat customer analysis
- Yearly and monthly cancellation trends

## Outcome
This analysis helps hotels understand customer behavior and reduce revenue loss due to cancellations.
